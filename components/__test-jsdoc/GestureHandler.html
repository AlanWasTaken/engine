Class: GestureHandler

GestureHandler

new GestureHandler(node, events)

Component to manage gesture events. Will track 'pinch', 'rotate', 'tap', and 'drag' events, on an as-requested basis.
Parameters:
Name Type Description
node Node The node with which to register the handler.
events Array An array of event objects specifying .event and .callback properties.
